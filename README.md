# Minda Code Landing

Landing page premium construida con Next.js 14 (App Router), TypeScript y TailwindCSS para la empresa Minda Code.

## Requisitos

- Node.js 18+
- npm 9+

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

La aplicación se abrirá en http://localhost:3000.

## Producción

```bash
npm run build
npm start
```

---

- Framework: Next.js 14 + App Router
- Estilos: TailwindCSS + componentes shadcn/ui personalizados
- Animaciones: framer-motion
- Iconos: lucide-react
